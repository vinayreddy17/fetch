In this task, I have fetched the data (get request)from the API https://poetrydb.org/author.
Then, I displayed the first 10 names using the "slice" function as buttons.
When you click on a button, the content displayed in the divs will be erased and the div with the poems of the author will be shown as buttons.
Again, when you click the poem of your choice, the same thing happens, and the poem will be displayed.
